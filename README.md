# 🛠️ Servicio Doméstico App - MVP

Proyecto Fullstack desarrollado como parte de mi portafolio profesional.

## 🚀 Tecnologías
- **Frontend:** React.js (Hooks, CSS moderno).
- **Backend:** Node.js + Express.
- **Arquitectura:** Monorepo.

## ⚙️ Cómo ejecutarlo localmente
1. Clona el repositorio.
2. Ve a `/backend`, ejecuta `npm install` y luego `node server.js`.
3. Ve a `/frontend`, ejecuta `npm install` y luego `npm start`.
